# Улучшенный сайт Roblox Scripts

## Описание
Этот сайт предоставляет коллекцию скриптов для Roblox с новогодним дизайном. После наших улучшений сайт стал более современным, красивым и интерактивным.

## Что нового

### Визуальные улучшения
- **Градиентные свечения**: Добавлены динамические градиентные эффекты с помощью CSS переменных
- **Улучшенные карточки скриптов**: Каждая карточка теперь имеет плавающие эффекты, тени и анимации при наведении
- **Улучшенная навигация**: Кнопки навигации теперь имеют анимированные подчеркивания и световые эффекты
- **Эффекты кода**: Блоки кода получили улучшенное оформление с градиентными подложками
- **Плавные анимации**: Все элементы плавно появляются при загрузке страницы

### Интерактивность
- **Hover-эффекты**: Все кнопки и карточки реагируют на наведение курсора
- **Световые эффекты**: Добавлены анимированные светодиодные эффекты для элементов
- **Параллакс-эффекты**: При прокрутке страницы элементы двигаются с разной скоростью

### Технические улучшения
- **CSS переменные**: Использованы CSS переменные для удобного управления цветами и эффектами
- **Современные анимации**: CSS-анимации обеспечивают плавные переходы
- **Адаптивный дизайн**: Все эффекты корректно работают на разных устройствах

## Технологии
- HTML5
- CSS3 (с использованием современных возможностей)
- JavaScript
- Tailwind CSS (для базовой стилизации)
- Font Awesome (для иконок)

## Особенности дизайна
- Новогодняя тематика с зимней цветовой палитрой
- Темный фон с декоративными элементами
- Светящиеся элементы и градиенты
- Плавающие иконки и текстовые элементы
- Современные UI-компоненты

## Установка
Сайт представляет собой статическую HTML-страницу, которую можно открыть напрямую в браузере или разместить на любом веб-сервере.
